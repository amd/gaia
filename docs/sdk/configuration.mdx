# Configuration

<Badge text="development" color="orange" />

## 16.1 Environment Variables

```bash
# LLM Configuration
export LEMONADE_BASE_URL="http://localhost:8000/api/v1"
export GAIA_API_DEBUG="true"
export GAIA_API_STREAMING="true"

# API Keys (for cloud providers)
export ANTHROPIC_API_KEY="sk-ant-..."
export OPENAI_API_KEY="sk-..."

# Logging
export GAIA_LOG_LEVEL="INFO"
```

---

## 16.2 Configuration in Code

```python
import os

# Read configuration
base_url = os.getenv("LEMONADE_BASE_URL", "http://localhost:8000/api/v1")
debug = os.getenv("GAIA_API_DEBUG", "false").lower() == "true"

# Use in agent
agent = MyAgent(
    base_url=base_url,
    debug_prompts=debug
)
```
