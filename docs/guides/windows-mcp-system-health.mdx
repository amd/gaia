**Agent-driven Windows system health check with GUI output.**

This example demonstrates GAIA's MCP client integration with the Windows MCP server, which provides tools for PowerShell execution and GUI automation. The agent collects system metrics (CPU, memory, disk), analyzes them with an LLM, and outputs a formatted health report directly into Notepad.

<Info>
**Source Code:** [`examples/mcp_windows_system_health_agent.py`](https://github.com/amd/gaia/blob/main/examples/mcp_windows_system_health_agent.py)
</Info>

## Prerequisites

- Windows 10/11
- Python 3.13+
- Node.js (for `uvx` command - auto-installs Windows MCP server)
- Lemonade server running (`lemonade-server serve`)

## Run the Example

```bash
uv run examples/mcp_windows_system_health_agent.py
```

## What to Expect

<Steps>
  <Step title="Agent connects to Windows MCP server">
    The agent auto-installs and connects to `windows-mcp` via `uvx`, which provides PowerShell and GUI automation tools.
  </Step>

  <Step title="Agent gathers system metrics">
    Using `mcp_windows_Shell`, the agent executes PowerShell commands to collect CPU usage, memory stats, and disk space.
  </Step>

  <Step title="LLM analyzes the data">
    The LLM reviews the metrics and creates a plain-text health report with findings and recommendations.
  </Step>

  <Step title="Report outputs to Notepad">
    The agent copies the report to clipboard via `mcp_windows_Shell` (Set-Clipboard), opens Notepad via `mcp_windows_Shell` (Start-Process), then pastes using `mcp_windows_Shortcut` (ctrl+v).
  </Step>
</Steps>
